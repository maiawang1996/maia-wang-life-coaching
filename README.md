# Maia Wang Life Coaching Website

A bilingual English and Chinese personal website for a life coach, built with Next.js, React, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes

- Default language is English.
- The language switcher updates all main site copy between English and Chinese.
- The contact form currently uses a front-end success message and can be connected to email or a backend later.
- Email, Red Note / Xiaohongshu, and YouTube links are configured in `components/LifeCoachSite.tsx`.
